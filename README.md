# MIDIFeatherController
MIDI Controller for Adafruit M4 Express with MIDI Feather and I2C OLED SSD1306

## Button setup:
* A2/D16: CANCEL
* A3/D17: LEFT
* A4/D18: RIGHT
* A5/D19: OK

## Versions

Adafruit CircuitPython 8.2.0 on 2023-07-05; Adafruit Feather M4 Express with samd51j19
